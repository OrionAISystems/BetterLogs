# Security Policy

## Supported Versions

The latest published release of `@trs/betterlogs` is the supported version for security fixes.

## Reporting A Vulnerability

If you believe you have found a security issue in BetterLogs, please do not open a public GitHub issue with exploit details.

Please report it privately by emailing `siphrcodez@gmail.com` with:

- a concise description of the issue
- affected versions
- reproduction steps or a proof of concept
- any suggested remediation if you have one

I will acknowledge reports as quickly as possible and work toward a fix and coordinated disclosure.
